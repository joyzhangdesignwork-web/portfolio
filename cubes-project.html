About

The Problem

Design feeds are noisy. I wanted a digest that was curated, summarized by AI, and delivered in a format I actually wanted to look at.

The System

Two Python scripts — one scrapes RSS feeds, one calls Claude to generate a daily report. The output feeds into a custom HTML frontend.

The Frontend

An animated GSAP cube grid titled "cube for your thoughts, joy" — because the interface should be as interesting as the content.
